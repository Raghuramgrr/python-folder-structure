# python-folder-structure
generator for python folder structure

python create_structure.py

# python : 
```
your_project_name/
├── docs/
├── src/
│   └── __init__.py
├── tests/
│   └── test_sample.py
├── venv/
├── .gitignore
├── README.md
└── requirements.txt
```

# maven: 
```
your_project_name/
├── src/
│   ├── main/
│   │   ├── java/
│   │   │   └── com/
│   │   │       └── example/
│   │   │           └── demo/
│   │   │               └── DemoApplication.java
│   │   └── resources/
│   │       └── application.properties
│   └── test/
│       ├── java/
│       │   └── com/
│       │       └── example/
│       │           └── demo/
│       │               └── DemoApplicationTests.java
│       └── resources/
├── target/
├── .gitignore
├── pom.xml
└── README.md
```
